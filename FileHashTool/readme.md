# 檔案 MD5 計算工具（Windows Forms）

這是一個以 Windows Forms 實作的 MD5 計算工具，支援多檔案並行計算、拖放加入、搜尋過濾、移除及一鍵複製結果。

## 功能概覽
- **檔案加入**：支援拖曳檔案到清單、或用「選擇檔案...」開啟多選檔案對話框。相同路徑的檔案（忽略大小寫）只會加入一次，並保留加入順序。
- **搜尋過濾**：輸入關鍵字即時篩選清單與輸出內容，只比對檔名（忽略大小寫）。
- **移除檔案**：可多選清單項目並點擊「移除選取檔案」刪除，同步移除對應的快取雜湊。
- **MD5 計算與輸出**：並行計算尚未快取的檔案 MD5，格式為 `檔案: {檔名}` 與 `MD5: {值}`，組與組之間自動空一行。運算錯誤時會顯示 `N/A (錯誤: …)`。
- **複製結果**：點擊「複製全部」即可將右側輸出文字放入剪貼簿。
- **進度顯示**：頂部標籤與進度條顯示目前運算進度；移除或重新觸發計算時會取消舊的運算。
- **鍵盤操作**：在清單按 `Ctrl+A` 可全選。

## 介面佈局
- 左側：檔案操作區（加入、移除、搜尋、檔案清單）。
- 右側：結果區（進度、複製全部按鈕、唯讀多行輸出文字框）。

## 執行方式
前置需求：.NET 10（Windows 桌面工作負載）。

```bash
dotnet run --project FileHashTool/FileHashTool.csproj
```

啟動後即可拖曳檔案或按下「選擇檔案...」開始使用。
